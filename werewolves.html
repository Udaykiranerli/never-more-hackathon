AI / ML Applications

Intelligent algorithms and predictive power

Explore the problem statements for the AI / ML Applications domain in the Never More Hackathon.

Problem Statements:

CEC-AIML-PS-01 - Spam Email Detection

Problem Statement

Develop a machine learning model to identify spam emails.

  • Email data collection
  • Text preprocessing
  • Spam vs non-spam classification

Background

Spam emails waste time and may contain security threats.

Expected Solution

An ML-based email filtering system with high accuracy.

CEC-AIML-PS-02 - Resume Keyword Matching

Problem Statement

Build a system to match resumes with job descriptions.

  • Resume parsing
  • Keyword extraction
  • Match score generation

Background

Manual resume screening is time-consuming and inefficient.

Expected Solution

An AI-powered tool that automates resume shortlisting.

CEC-AIML-PS-03 - Movie Recommendation System

Problem Statement

Create a recommendation system for suggesting movies to users.

  • User preference analysis
  • Content or collaborative filtering
  • Personalized recommendations

Background

Users struggle to find relevant content among vast movie libraries.

Expected Solution

An ML-based recommendation engine for personalized movie suggestions.

CEC-AIML-PS-04 - Fake News Classification

Problem Statement

Develop a model to classify news as real or fake.

  • Text analysis
  • Feature extraction
  • Binary classification

Background

Fake news spreads misinformation and influences public opinion.

Expected Solution

An NLP-based fake news detection system.

CEC-AIML-PS-05 - Student Result Prediction

Problem Statement

Build a predictive model to forecast student academic results.

  • Historical data analysis
  • Feature selection
  • Performance prediction

Background

Early prediction helps in providing timely academic support.

Expected Solution

An ML model that predicts student outcomes and identifies risk factors.

CEC-AIML-PS-06 - FAQ Chatbot

Problem Statement

Develop an AI-powered chatbot to answer frequently asked questions automatically.

  • Natural language understanding
  • Intent detection
  • Automated responses

Background

Handling repetitive queries manually is time-consuming and inefficient.

Expected Solution

An NLP-based chatbot that provides instant and accurate responses to user queries.
