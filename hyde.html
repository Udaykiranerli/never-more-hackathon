Internet of Things

Connected devices and smart automation

Explore the problem statements for the Internet of Things domain in the Never More Hackathon.

Problem Statements:

CEC-IOT-PS-01 - Smart Energy Monitoring

Problem Statement

Develop an IoT-based system to monitor energy consumption in real time.

  • Sensor-based data collection
  • Real-time monitoring
  • Usage analytics

Background

Unmonitored energy usage leads to wastage and high operational costs.

Expected Solution

An IoT-enabled energy monitoring system with dashboards and alerts.

CEC-IOT-PS-02 - Water Level Monitoring

Problem Statement

Build a system to monitor water levels in tanks or reservoirs.

  • Level sensors
  • Real-time alerts
  • Overflow and shortage detection

Background

Water overflow and scarcity often go unnoticed due to lack of monitoring.

Expected Solution

An IoT-based water level monitoring and alert system.

CEC-IOT-PS-03 - Smart Parking System

Problem Statement

Design a system to detect and display parking slot availability.

  • Vehicle detection sensors
  • Real-time slot status
  • User display interface

Background

Finding parking spaces in urban areas is time-consuming and inefficient.

Expected Solution

A smart parking solution using IoT sensors and live updates.

CEC-IOT-PS-04 - Weather Monitoring Station

Problem Statement

Create an IoT station to collect and display weather data.

  • Temperature, humidity, rainfall sensing
  • Continuous data logging
  • Visualization dashboard

Background

Localized weather data is often unavailable or inaccurate.

Expected Solution

An IoT-based weather monitoring system with real-time visualization.

CEC-IOT-PS-05 - Health Monitoring System

Problem Statement

Develop a system to monitor basic health parameters remotely.

  • Heart rate and temperature sensing
  • Real-time data transmission
  • Alert generation

Background

Continuous health monitoring is essential for early risk detection.

Expected Solution

An IoT-based health monitoring platform with alerts and data tracking.

CEC-IOT-PS-06 - Smart Streetlight Automation

Problem Statement

Build an automated streetlight control system using IoT.

  • Light intensity sensing
  • Automatic ON/OFF control
  • Energy efficiency

Background

Manual streetlight operation leads to energy wastage and maintenance issues.

Expected Solution

An IoT-enabled smart streetlight system that optimizes energy usage automatically.
