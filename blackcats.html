Smart Agriculture

Innovative solutions for sustainable farming

Explore the problem statements for the Smart Agriculture domain in the Never More Hackathon.

Problem Statements:

CEC-SAG-PS-01 - Crop Advisory Web Application

Problem Statement

This problem statement proposes the development of a smart crop advisory system to assist farmers in selecting suitable crops, fertilizers, and sowing seasons based on local conditions. The system can:

  • Accept inputs such as soil type, location, and season
  • Recommend suitable crops and fertilizers
  • Provide basic weather-based suggestions
  • Display best farming practices
  • Support regional language interfaces

Background

Farmers often rely on traditional knowledge or guesswork while selecting crops, leading to low yield and losses. Lack of accessible digital advisory platforms worsens the problem, especially in rural areas.

Expected Solution

A web or mobile-based advisory platform that includes:

  • Crop and fertilizer recommendation engine
  • Weather integration
  • Simple user-friendly interface
  • Localized language support

CEC-SAG-PS-02 - Leaf Disease Image Classifier

Problem Statement

Develop an image-based crop disease detection system to identify plant leaf diseases at early stages. The system can:

  • Accept leaf images from users
  • Classify diseases using AI models
  • Suggest preventive measures
  • Display confidence levels
  • Work with limited datasets

Background

Crop diseases significantly reduce productivity. Farmers lack immediate access to experts, resulting in delayed treatment.

Expected Solution

An AI-powered image classification tool using:

  • Pre-trained CNN models
  • Simple upload interface
  • Disease database and remedies

CEC-SAG-PS-03 - Smart Irrigation Decision Support System

Problem Statement

Design a system to optimize irrigation schedules based on environmental and soil data. The system can:

  • Analyze moisture and weather data
  • Suggest irrigation timing
  • Reduce water wastage
  • Display usage insights

Background

Over-irrigation leads to water wastage and soil degradation. Many farmers lack scientific irrigation guidance.

Expected Solution

A decision-support dashboard using simulated or sensor data.

CEC-SAG-PS-04 - Farmer Market Price Tracker

Problem Statement

Build a platform that provides farmers with updated crop market prices. The system can:

  • Fetch daily prices
  • Display trends
  • Compare markets
  • Enable informed selling decisions

Background

Farmers often sell produce without knowing real-time prices, resulting in financial losses.

Expected Solution

A web/mobile app integrating public datasets or APIs.

CEC-SAG-PS-05 - Fertilizer Recommendation Tool

Problem Statement

Develop a rule-based fertilizer recommendation system.

  • Suggest fertilizers based on crop and soil
  • Avoid overuse
  • Improve yield

Background

Improper fertilizer usage harms soil health and crops.

Expected Solution

A simple rule-based decision system with a clean UI.

CEC-SAG-PS-06 - Weather Alert System for Farmers

Problem Statement

Create a weather alert system to notify farmers of adverse conditions.

  • Provide rainfall and temperature alerts
  • Issue warnings
  • Support SMS/app notifications

Background

Sudden weather changes damage crops due to lack of timely alerts.

Expected Solution

An alert-based system using weather APIs.
